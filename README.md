## Getting Started

Install modules:

```bash
npm i
# or
yarn
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Testing

```bash
npm run cypress:open
# or
yarn cypress:open
```
